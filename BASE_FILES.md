# BASE_FILES.md — Canonical Source of Truth for EDAP Model

## Purpose
This file defines the **minimum set of files** from which an AI assistant or collaborating researcher can reconstruct the entire EDAP project — code, article, figures, tables, and CI pipeline.

These files are the **canonical source of truth**. All other files in the repository are either **generated** from these or are **auxiliary** (documentation for humans, CI configuration).

This file itself is NOT a base file. It is a meta-document describing the system from outside.

## Generated Files (Reproducible)

All other files are generated from the base files by following the build pipeline in `INTERFACES.md`:

| Category | Files | Generated by |
|----------|-------|-------------|
| Model code | `edap_model/*.py` | ADR equations + INTERFACES contracts + parameters.json |
| Figures | `figures/*.py` → `output/fig*.png` | THESES descriptions + INTERFACES contracts + data |
| Numerical results | `output/results.json` | `python run_model.py` |
| LaTeX tables | `output/table_*.tex` | `latex_export.py` ← results.json |
| Compiled article | `paper/article.pdf` | `pdflatex` ← article.tex + sections + tables + figures |
| CI pipeline | `.github/workflows/release.yml` | INTERFACES build pipeline |
| CI scripts | `.github/scripts/*.py` | GUARDRAILS + INTERFACES |

## Reconstructing the Project

An AI assistant or researcher with these 22 base files can reproduce every artefact:

1. **Model code** — from ADR equations + INTERFACES contracts + parameters
2. **Calibration** — from ADR-004 + INTERFACES GPU path + parameters
3. **Metrics** — from ADR-011 + ADR-012 + ADR-013
4. **Normalizer** — from ADR-002 + normalization_params.json
5. **Figures** — from THESES descriptions + INTERFACES contracts + data
6. **Export and tables** — from INTERFACES pipeline + INDEX descriptions
7. **Article sections** — from THESES + GUARDRAILS + numerical results
8. **Article skeleton** — standard LaTeX preamble + `\input{sections/...}`
9. **CI pipeline** — from INTERFACES build pipeline

## Verification

After reconstruction, all three must pass:

```bash
python run_model.py
python -m pytest tests/test_model.py -v
python .github/scripts/guardrail_check.py abstract paper/sections/ GUARDRAILS.md output/results.json
```
